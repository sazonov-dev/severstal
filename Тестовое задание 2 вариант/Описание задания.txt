Перед вами проект написанный на React - TypeScript.

Проект представляет собой иерархическую таблицу с предварительно подготовленными данными.

Необходимо:

1. Провести рефакторинг кода в компонентах "App" и "ParentChildTable".
2. Добавить сортировку по клику на заголовки столбцов name, balance и isActive без использовании возможностей таблицы.
3. Добавить боковую панель в которой, по клику на любую ячейку, будет отображаться информация о пользователе.

Бонусное задание: Написать unit тесты с использованием "Jest" для компонента "ParentChildTable".
